# Markdown2Word

AIで作成したMarkdownテキストをWordファイル（.docx）に一発変換するツールです。

---

## 使い方

### 基本手順

1. `Markdown2Word_Pro.exe` を起動する
2. AIのチャット画面でMarkdownをコピーする
3. 「クリップボード貼り付け」ボタンをクリック
4. テンプレートを選択する（後述）
5. 出力ファイル名・保存先を指定する
6. 「Wordに変換」ボタンをクリック

### ファイルから読み込む場合

- 「ファイルから読み込み」ボタンで `.md` や `.txt` ファイルを選択
- またはファイルを `Markdown2Word_Pro.exe` に**ドラッグ&ドロップ**（全自動で変換してWordが開く）
  - 対応拡張子: `.md`, `.txt`, `.markdown`
  - 文字コード: UTF-8, UTF-8（BOM付き）, Shift_JIS に自動対応しています。

---

## 便利な自動補正機能

Markdownの構文エラーでよくある**「見出し記号（#）の後のスペース忘れ」を自動で修正**してから変換します。
（例：`#見出し` → `# 見出し` に自動補正）

---

## テンプレートは必須ではありません

**テンプレートがなくても、そのまま使えます。**

テンプレートの有無で動作は変わりません。違いはデザインだけです。

| 状態 | 変換結果 | こんな人向け |
|------|---------|------------|
| テンプレートなし | 游明朝・青見出しなどが自動適用される | とにかく今すぐ使いたい |
| テンプレートあり | 会社指定のデザインが自動適用される | フォーマットを統一したい |

まずはテンプレートなしで試してみて、「デザインも整えたい」と思ったときにテンプレートを作るのがおすすめです。

---

## 自動適用されるスタイル（テンプレートなし時）

テンプレートを選択しない場合でも、以下のスタイルがプログラムによって自動で上書き適用されます。

| Wordのスタイル | 対応するMarkdown | 自動設定 |
|--------------|-----------------|---------|
| 標準          | 本文テキスト     | 游明朝（またはMS 明朝）・10.5pt |
| 見出し 1      | `# テキスト`    | 青色・太字・18pt |
| 見出し 2      | `## テキスト`   | 紺色・太字・14pt |
| 見出し 3      | `### テキスト`  | 黒・太字・12pt |
| 表            | テーブル        | Light Shading Accent 1 |

## テンプレート（スタイルファイル）の作り方

テンプレートとは「フォント・色・サイズなどのデザインのお手本」となるWordファイルです。
一度作っておくと、変換するたびに会社指定のフォーマットが自動適用されます。

### 手順

**① 普通にWordを開いて、スタイルを設定する**

Wordの「ホーム」タブ →「スタイル」から各スタイルを右クリック →「変更」で編集します。

> ※ スタイル名は**必ずこの名前**（見出し 1、見出し 2 など）のままにしてください。
> 名前を変えると変換後に反映されません。

**② ダミーの内容を書いておく（推奨）**

変換結果のイメージを確認しやすくなります。実際の文章は不要で、スタイルを当てたサンプルテキストだけで構いません。

**③ 名前を付けて保存する**

- `style.docx` → デフォルトのテンプレートとして使用されます
- `templates/報告書.docx` → テンプレート選択に「報告書」として表示されます
- `templates/議事録.docx` → テンプレート選択に「議事録」として表示されます

---

## フォルダ構成

```
Markdown2Word_Pro.exe     ← 実行ファイル
style.docx                ← デフォルトテンプレート（任意）
templates/                ← 複数テンプレートを置くフォルダ（任意）
    報告書.docx
    議事録.docx
    提案書.docx
```

> `templates` フォルダは自分で作成してください。

## exeのビルド方法（開発者向け）

### 必要なもの

- Python 3.x
- Pandoc（https://pandoc.org/installing.html）
  ※ pandoc-3.x.x-windows-x86_64.msi を選ぶ

- 必要なPythonライブラリ

```
pip install pypandoc pyinstaller python-docx customtkinter
```

### ビルドコマンド

`Markdown2Word_Master.py` があるフォルダで以下を実行：

```
python build_master.py
```

`dist/Markdown2Word_Pro.exe` が生成されます。

**`build_master.py` が自動でやること：**

- 古い `build/` `dist/` フォルダを削除してクリーンビルド
- `customtkinter`・`python-docx` の必須データファイルを同梱
- `templates/` フォルダが存在すればexe内に自動同梱
- `style.docx` が存在すればexe内に自動同梱

---

## 注意事項

- **Pandocが必要です。** 配布先のPCにもPandocがインストールされている必要があります。
  インストーラー：https://pandoc.org/installing.html
- 出力先フォルダを指定しない場合は **Downloadsフォルダ** に保存されます。
- 画像はMarkdownファイルからの相対パス・絶対パス・URLが使えます。
